什么是ICP经营性许可证？

ICP全称Internet Content Provider，即向广大用户综合提供互联网信息业务和增值业务的电信运营商，其必须具备的证书即为ICP经营性许可证。根据国家《互联网管理办法》规定，经营性网站必须办理ICP经营性许可证。

ICP经营性许可证与ICP备案有什么区别，什么是“三证”？

很多用户经常将ICP备案和ICP经营性许可证混为一谈，实际上这两者有很大区别:ICP备案仅是信息产业部对网站的一种管理，门槛较低，只要登陆工信部官网录入网站真实信息后经审核即可获得，目的是为了防止在网上从事非法的网站经营活动。而ICP经营性许可证申请门槛比较高，需要特定条件并提交相关申办材料，审核严格，获得ICP经营性许可证才是经营性企业网站规范合法经营的重要标志。

所谓“三证”，ICP备案信息（ICP备）来自工业和信息化部ICP/IP备案管理系统，增值电信业务经营许可证（ICP许可证）来自工业和信息化部电信业务市场综合管理信息系统，公安机关网安备案信息（网安备）来自全国公安机关互联网安全管理平台 。获得ICP许可证、甚至集齐“三证”，说明互联网金融平台达到了最高的监管合规要求。
